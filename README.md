 # 📝 Notes App

A simple and responsive Notes App built using React and Tailwind CSS.  
This app allows users to create, manage, and delete notes easily.  
All notes are stored in the browser using localStorage, so data remains even after refreshing the page.

---

## 🚀 Live Demo
👉 https://note-app-swart-nine.vercel.app

---

## ✨ Features

- 📝 Add new notes  
- ❌ Delete notes  
- 💾 Data stored using localStorage  
- 📱 Fully responsive design  
- ⚡ Fast and lightweight
